# Sensor Interfacing Embedded Systems

## Overview
This project demonstrates basic sensor interfacing using embedded systems. It includes reading real-time data from sensors and controlling outputs based on sensor values.

---

## Components Used
- Arduino / Raspberry Pi
- Temperature Sensor (DHT11/DHT22) or LDR
- LED
- Resistors
- Breadboard

---

## Functionality
- Reads sensor data in real time
- Displays output in serial monitor
- Controls LED based on threshold condition

---

## Example Logic
- If light level is low → LED ON  
- If light level is high → LED OFF
